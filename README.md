# 🤖 AI Learning Hub

A modern landing page built using HTML and CSS. This project showcases an AI and Frontend learning platform with a clean, responsive design.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern Dark Theme UI
- 🧭 Responsive Navigation Bar
- 🖼 Hero Section
- 📚 Skills Section
- 🃏 Interactive Course Cards
- 📩 Footer with Contact Information
- ✨ Smooth Hover Effects

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Google Fonts (Montserrat)
